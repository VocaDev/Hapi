# Hapi · Jury Package

Open `index.html` directly in a browser to run the high-fidelity clickable prototype. Open `pitch-deck.html` for the 10-slide jury deck; use arrow keys, PageUp/PageDown, Home, and End.

## Recommended Demo Flow

1. Start in `index.html` on **Paneli** and say the core line: “Një ngjarje, gjithë shteti përgjigjet.”
2. Click **⚡ Simulo: ngjarja vjen vetë te qytetari** — the state acts first: toast, bell pop, new notification, dashboard counters update, and Hapi auto-opens **Lindja e Artës**.
3. Walk through Lirie giving birth to Arta in Prizren and point out **Only ask when needed**: she confirms IBAN, QMF, contact, and consent, but is not asked for data the state already has. Click **→ Ku shkojnë të dhënat e mia?** to surface the data-sharing map on demand.
4. Open **Asistenti AI** and show Albanian-first guidance with guardrails.
5. Open **Dokumentet** and show the official-looking certificate, e-stamp, SHA fingerprint, and append-only document trail (covers the trust/audit story without a dedicated audit screen).
6. Open **Intake & raste** to show the government-side case-management dashboard.
7. Briefly visit **Biznesi** and **18 vjeç** to prove Hapi generalizes across citizen-initiated and automatic triggers.
8. Present `pitch-deck.html` for the structured 3-minute jury pitch.

## Files

- `index.html` — clickable product prototype covering citizen and government surfaces.
- `pitch-deck.html` — 10-slide 16:9 HTML deck with keyboard navigation.
- `DEMO_SCRIPT.md` — 3-minute presenter script.
- `REQUIREMENT_COVERAGE.md` — traceability map from major FR/NFR groups to prototype and deck surfaces.
- `HAPI_APP_REQUIREMENTS.md` — source requirements brief.

## Positioning

Hapi is not a replacement for eKosova. It is a proactive life-event orchestration layer inside or beside eKosova: detection, journey creation, institution routing, document generation, notifications, deadline protection, AI guidance, and audit transparency.
